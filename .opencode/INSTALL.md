# Installing Code Factory for OpenCode

1. Clone the repository:
   ```bash
   git clone https://github.com/xmtplabs/code-factory.git
   ```

2. Add the plugin to your `opencode.json`:
   ```json
   {
     "plugins": [
       {
         "name": "code-factory",
         "path": "/path/to/code-factory"
       }
     ]
   }
   ```
